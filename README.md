## 📖 Overview

This project analyzes the impact of AI R&D investments on revenue growth and stock market performance across leading technology companies using time-series financial data.

Exploratory Data Analysis (EDA) and trend identification were performed to uncover relationships between investment patterns and market responses. The analysis and trend comparison were carried out using Python, while Microsoft Excel was used for initial data validation.

## 🎯 Business Problem

This project aims to:

* Identify the amount spent by companies on AI Research & Development (R&D)
* Determine revenue earned by companies
* Analyze date-wise impact on stock prices
* Identify events with maximum stock impact
* Determine AI-driven revenue growth
* Investigate correlations among financial variables
* Compare R&D expenditure vs revenue year by year
* Analyze event-based stock impacts
* Evaluate changes in market indices across companies and years

## ▶️ How to Run This Project

### Prerequisites

Ensure the following are installed on your system:

* Python 3.8 or above
* Anaconda (Optional)
* Jupyter Notebook or Jupyter Lab

### Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Project Setup

1. Download or clone the repository.
2. Place the dataset file `ai_financial_market_data.csv` in the same directory as the notebook.
3. Launch Jupyter Notebook or Jupyter Lab.
4. Open:

```text
ai's_financial_market_trends_analysis.ipynb
```

5. Run all cells to reproduce the analysis and visualizations.
